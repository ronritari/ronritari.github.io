About the Project

FutStats is a passion project designed to explore and visualize football (soccer) statistics. This project demonstrates my interest in combining web development with data analytics to create meaningful insights from sports data.

Features

  • Player statistics and performance metrics
  • Team analysis and comparison
  • Interactive data visualizations
  • Real-time match updates
